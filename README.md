# num_calendar

Num Calendar Application

## Prerequisite

- fvm
- git
- xcode or android sdk

## Build & Run

```bash
cd num_calendar
```

```bash
fvm flutter pub get
```

```bash
fvm flutter run
```

> Used flutter version is `3.19.6`
